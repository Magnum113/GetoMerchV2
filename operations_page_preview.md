# 📊 Операционный центр - Предпросмотр

## Как выглядит страница Operations

### 🎯 Основной заголовок
```
Операционный центр
Ежедневный план: что отправить, произвести и заказать
[Кнопка: Пересчитать статусы]
```

### 📦 1. Блок "К отправке сегодня" (READY_TO_SHIP)
```

┌─────────────────────────────────────────────────────────────┐
│ Заказ #12345                                         5000₽ │
```

## 🎨 Дизайн и стили

- **Цветовая схема**: 
  - Готово к отправке: 🟢 зеленый
  - Нужно произвести: 🔵 синий
  - Дефицит материалов: 🟠 оранжевый
  - Заявка на пополнение: 🔴 красный

- **Макет**: 2x2 сетка (первый блок занимает всю ширину)
- **Анимации**: Кнопка пересчета имеет анимацию вращения при загрузке
- **Отзывчивость**: Адаптивный дизайн для разных размеров экрана

## 🔧 Функциональность

1. **Автоматическая загрузка**: Данные загружаются при открытии страницы
2. **Пересчет статусов**: Кнопка обновляет все операционные статусы
3. **Действия**: Кнопки для отправки заказов и запуска производства
4. **Реальное время**: Показывает текущее время последнего обновления
